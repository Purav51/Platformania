The game is made using Unity 6000.3.9f1 version.
All the code can be found within the scripts forlder located in the Assets folder.
The Game folder contains the latest build for the game, to play it simply download the zip and open the game folder and run Platformania.exe.
the controls for the game are simple use a and d to move right and left, w and s to move up and down the ladders, space to jump and left click to fire projectiles at the enemies.
